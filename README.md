This website serves as the digital storefront for Prestige Barbershop (Everett, MA). The goal of the site is to drive conversions (appointments) towards the barbers' individual booking pages, as well as to showcase the brand's high-quality cuts and premium aesthetic.

### Key Objectives:

- Performance: Sub-second load times and smooth transitions.
- SEO: Optimized meta tags and semantic HTML for local search visibility.
- UX: Intuitive navigation, mutilingual accessibility, mobile-first design, and interactive elements (custom intro animation).
